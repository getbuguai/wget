# wget
一个简易的 wget 下载小工具。

## 背景

windows 上 命令行不能下载文件，或者我不知道怎么下载。

于是就写了一个简单的下载工具。

有好的建议可以提出来，[进行改进。](https://github.com/getbuguai/wget/issues)

## 构建

```
git clone https://github.com/getbuguai/wget.git
cd wget && go build 

# 以工具包的形式, 会自动安装到 GOBIN
go get -u github.com/getbuguai/wget

```
